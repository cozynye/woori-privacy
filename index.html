개인정보처리방침

시행일: 2026년 3월 17일

1. 수집하는 개인정보

우리(Woori) 앱은 다음 정보를 처리합니다:

  • Apple ID — 로그인 및 iCloud 동기화 용도
  • 사용자 입력 데이터 — 일정, 체크리스트, 피드(사진/글), 기념일, 위시리스트, 한마디 메모
  • 기기 타임존 — 시차 표시 기능 용도

2. 개인정보의 저장

모든 데이터는 사용자의 iCloud 계정(Apple CloudKit)에 저장됩니다.

  • 별도의 외부 서버를 운영하지 않습니다
  • 개발자는 사용자 데이터에 접근할 수 없습니다
  • 데이터는 Apple의 보안 인프라에 의해 보호됩니다

3. 데이터 공유

앱 내에서 커플로 연결된 정확히 2명의 사용자 간에만 데이터가 공유됩니다.

  • 제3자에게 데이터를 제공하지 않습니다
  • 광고 SDK를 사용하지 않습니다
  • 분석/추적 도구를 사용하지 않습니다

4. 데이터 삭제

  • 앱을 삭제하면 기기의 로컬 데이터가 삭제됩니다
  • iCloud 데이터는 설정 > Apple ID > iCloud > 저장 공간 관리에서 직접 삭제할 수 있습니다

5. 아동 보호

본 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

6. 변경 사항

본 방침이 변경될 경우 앱 업데이트를 통해 안내드립니다.

7. 문의

Privacy Policy

Effective: March 17, 2026

1. Information We Collect

Woori processes the following information:

  • Apple ID — For login and iCloud synchronization
  • User-generated content — Calendar events, checklists, feed posts (photos/text), anniversaries, wishlists, and one-liner memos
  • Device timezone — For timezone display feature

2. Data Storage

All data is stored in the user's iCloud account (Apple CloudKit).

  • We do not operate any external servers
  • The developer cannot access user data
  • Data is protected by Apple's security infrastructure

3. Data Sharing

Data is shared only between exactly 2 users connected as a couple within the app.

  • We do not share data with third parties
  • We do not use advertising SDKs
  • We do not use analytics or tracking tools

4. Data Deletion

  • Deleting the app removes local data from the device
  • iCloud data can be deleted via Settings > Apple ID > iCloud > Manage Storage

5. Children's Privacy

This app does not knowingly collect personal information from children under 14.

6. Changes

Any changes to this policy will be communicated through app updates.

7. Contact

Privacy inquiries: cozynye@gmail.com
